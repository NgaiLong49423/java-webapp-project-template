# GitHub Project Fields Template

| Field Name | Type | Options | Purpose |
|---|---|---|---|
| Status | Single Select | Backlog, Ready, In Progress, Code Review, Testing, Blocked, Done | Track workflow state. |
| Priority | Single Select | High, Medium, Low | Track importance. |
| Layer | Single Select | Database, Backend, Frontend, Integration, Testing, Documentation, DevOps | Track technical area. |
| Story Point | Number | 1, 2, 3, 5, 8 | Track estimated effort. |
| Sprint | Single Select | Sprint 1, Sprint 2, Sprint 3, Sprint 4, Sprint 5 | Track delivery phase. |
| Module | Single Select | Authentication, Booking, Service, Membership, Payment, Reporting | Track business module. |
| Complexity | Single Select | Easy, Medium, Hard | Track implementation difficulty. |
